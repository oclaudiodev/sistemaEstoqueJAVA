## 📦 Sistema de Controle de Estoque em Java

Projeto desenvolvido para praticar conceitos de **Programação Orientada a Objetos (POO)** utilizando Java, simulando um sistema simples de gerenciamento de estoque no terminal.

### 🚀 Conceitos aplicados no projeto

* **Abstração**

  * Criação da classe abstrata `Produto` como modelo base para os produtos.

* **Herança**

  * Classes `Caneta` e `Lapis` herdando atributos e comportamentos da classe `Produto`.

* **Polimorfismo**

  * Uso de um array de `Produto` permitindo armazenar diferentes tipos de produtos no estoque.

* **Encapsulamento**

  * Atributos privados com acesso controlado através de métodos `get` e `set`.

* **Sobrescrita de métodos (Override)**

  * Implementação específica dos métodos abstratos nas classes filhas.

* **Estrutura de dados (Array de Objetos)**

  * Armazenamento e gerenciamento de produtos dentro do estoque.

* **Manipulação de estoque**

  * Adicionar produtos
  * Remover produtos
  * Listar produtos cadastrados
  * Adicionar quantidade ao produto
  * Remover quantidade do produto

### 🧠 Objetivo do projeto

Praticar **lógica de programação, modelagem de classes e aplicação de conceitos fundamentais de POO em Java**, simulando um sistema básico de controle de estoque.
